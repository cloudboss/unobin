# Cloudboss libraries

Cloudboss publishes Unobin libraries in separate repositories. Each library has its own manual.

| Library | Import path | Docs | Repository |
| --- | --- | --- | --- |
| Standard library | `github.com/cloudboss/unobin-library-std` | [Docs](https://cloudboss.co/docs/unobin-library-std) | [Repository](https://github.com/cloudboss/unobin-library-std) |
| AWS library | `github.com/cloudboss/unobin-library-aws` | [Docs](https://cloudboss.co/docs/unobin-library-aws) | [Repository](https://github.com/cloudboss/unobin-library-aws) |
| Namecheap library | `github.com/cloudboss/unobin-library-namecheap` | [Docs](https://cloudboss.co/docs/unobin-library-namecheap) | [Repository](https://github.com/cloudboss/unobin-library-namecheap) |
